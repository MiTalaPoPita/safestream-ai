 SafeStream AIReal-time Photosensitive Protection & Flicker AnalysisSafeStream AI is a cutting-edge, client-side video player designed to protect users with photosensitive epilepsy. By utilizing advanced luminance analysis and real-time filtering, the system detects dangerous flicker frequencies and applies "Digital Sunglasses" to mitigate risks instantly.🚀 Key FeaturesReal-time Detection: Analyzes video frames on-the-fly to detect flickering between 3Hz and 30Hz.Digital Sunglasses: An intelligent AI-driven filter that automatically adjusts brightness, contrast, and blur when a risk is detected.ITU-R BT.1702 Compliant: Built based on global standards for reducing photosensitive epileptic seizures.Privacy-First (Zero Upload): All processing happens locally in the browser. Your videos never touch a server.FFmpeg-Powered Export: Bake the safety filters directly into the video file using FFmpeg.wasm for safe sharing.🛠️ Tech StackFramework: Next.js 16 (App Router)Processing: Web Workers for non-blocking luminance analysis.Video Engine: FFmpeg.wasm (Multi-threaded)Styling: Modern Glassmorphism UI with CSS Variables.Deployment: Vercel with Cross-Origin Isolation headers.🧬 How it WorksFrame Capture: The system captures video frames at high speed using a hidden canvas.Luminance Analysis: Each frame is analyzed to calculate its relative luminance.Risk Assessment: We calculate the frequency ($Hz$) and intensity of luminance changes.Mitigation: If the thresholds are exceeded, the Digital Sunglasses component activates an overlay that lowers the risk Level to SAFE.📥 Getting StartedClone the repository: git clone https://github.com/your-username/safestream-ai.gitInstall dependencies: npm installRun locally: npm run devNote: Requires a browser that supports SharedArrayBuffer (Chrome/Edge/Firefox) for FFmpeg features.⚖️ LicenseDistributed under the MIT License.
